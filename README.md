# diamond

## Feature

a basic UI component base on React

## Install

install from github

```
npm install git+https://github.com/sgnilo/diamond.git
```

install from npm

```
npm install diamond
```

## Usage

```
import { ImgPreview } from 'diamond'

const url = ''

ReactDom.render(<ImgPreview url={url}><img src={ur}/></ImgPreview>, mountNode)
```
